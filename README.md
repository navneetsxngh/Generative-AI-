# Generative AI using LangChain

A collection of hands-on projects and notebooks exploring Generative AI concepts using the [LangChain](https://www.langchain.com/) framework. This repository covers a wide range of topics — from building simple chatbots to advanced RAG pipelines and agent-based systems.

## 📁 Projects

| Folder | Description |
|---|---|
| `Simple GenAI Application` | A basic generative AI app to get started with LangChain |
| `Chatbot using Langchain` | A conversational chatbot built with LangChain |
| `Conversation Q&A` | Q&A system with conversational memory |
| `Chat SQL` | Natural language interface to query SQL databases |
| `PDF Query RAG using Langchain` | Retrieval-Augmented Generation (RAG) pipeline for querying PDF documents |
| `Text Summarization` | Summarize long texts using LLMs via LangChain |
| `Hybrid Search` | Combines dense and sparse retrieval for improved search accuracy |
| `Tools and Agents` | LangChain agents with custom tools for autonomous task execution |
| `LCEL GenAI App` | App built using LangChain Expression Language (LCEL) |
| `GraphDB with Langchain` | Integrating graph databases with LangChain |
| `Langchain x Huggingface` | Using Hugging Face models with LangChain |
| `Fine Tuning` | Fine-tuning LLMs for custom tasks |
| `Langchain Prerequities` | Prerequisite concepts and setup for working with LangChain |

## 🛠️ Tech Stack

- **Python** (3.x)
- **LangChain**
- **Hugging Face**
- **Jupyter Notebooks**
- **uv** (dependency management)

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/navneetsxngh/Generative-AI-using-Langchain.git
   cd Generative-AI-using-Langchain
   ```

2. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

3. **Open any notebook** in your preferred environment (Jupyter, VS Code, etc.) and follow the instructions inside.
